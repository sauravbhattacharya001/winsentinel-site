Pricing FAQ

Does WinSentinel send anything to the cloud or use a cloud LLM?

No. Every audit module is deterministic .NET code hitting real Windows APIs — zero network calls. The optional natural-language advisor talks to a local Ollama instance (default llama3 on localhost:11434) and falls back to a rule-based engine when Ollama isn't installed. No OpenAI, Anthropic, or any other third-party LLM is ever called. Safe for air-gapped, Essential 8 / CIS / SOC2-regulated environments.

Is the CLI really free forever?

Yes. The CLI and every audit module are MIT-licensed on GitHub and published to NuGet. We never paywall an audit check itself — Pro is about workflow features (monitoring, scheduling, reports) on top of what the free tool already finds.

What happens if my Pro license expires offline?

Pro features keep working for 30 days without an internet check-in. After that, the CLI falls back to Free-tier behavior — your historical reports stay on disk and the audit modules keep running; you just lose monitoring, scheduling, and PDF export until you reconnect.

Is the license per-user or per-machine?

Per-user for Pro. You can activate a Pro license on every Windows box you personally use (desktop, laptop, VM), though we only count one concurrent monitor daemon. Team is per-organization, up to 10 active machines reporting to the fleet dashboard.

Do you store my scan results?

Not on Free, never. On Pro, scan results stay on your machine — only an opt-in crash report ever leaves. On Team, only the machines you explicitly connect to the fleet dashboard push results, and you control what's redacted before upload.

Can I get a refund?

Yes — 30-day, no-questions-asked refund on Pro and Team. Cancel directly in Stripe (link in your purchase receipt) and we'll auto-refund the last charge.

When does billing actually start?

Stripe Checkout is being wired up right now. If you click “Start trial” before launch, you'll go straight to the waitlist and get a one-time invite when Pro opens.
